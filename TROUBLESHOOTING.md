# Решение на проблеми

## Проблем: Сайтът не зарежда

### Решение 1: Проверка на Node.js версията

Текущата версия на Node.js е 16.14.1, което може да причини проблеми. Препоръчително е да обновите до Node.js 18 или по-нова версия.

**За Windows:**
1. Изтеглете Node.js 18 LTS от https://nodejs.org/
2. Инсталирайте го
3. Рестартирайте терминала
4. Изпълнете `npm install` отново
5. Стартирайте `npm run dev`

### Решение 2: Изчистване на кеша

```bash
# Изтриване на node_modules и package-lock.json
Remove-Item -Recurse -Force node_modules, package-lock.json

# Преинсталиране
npm install

# Стартиране
npm run dev
```

### Решение 3: Проверка на порта

Ако порт 3000 е зает, Vite автоматично ще използва друг порт. Проверете конзолата за съобщение като:
```
Local:   http://localhost:5173/
```

### Решение 4: Проверка на браузъра

1. Отворете Developer Tools (F12)
2. Проверете конзолата за грешки
3. Проверете Network tab за неуспешни заявки

### Решение 5: Временно изключване на PWA

Ако проблемът продължава, временно изключете PWA плагина:

В `vite.config.ts`, коментирайте VitePWA:
```typescript
plugins: [
  react(),
  // VitePWA({ ... })
]
```

## Проблем: Грешки с Supabase

Ако виждате грешки свързани с Supabase, проверете:
1. Да сте изпълнили SQL скрипта в Supabase
2. Да сте създали Storage bucket `documents`
3. Да сте направили bucket-а публичен

## Проблем: Грешки с импорти

Ако имате проблеми с импорти, проверете:
1. Всички зависимости са инсталирани: `npm install`
2. TypeScript конфигурацията е правилна
3. Всички файлове са на правилните места
