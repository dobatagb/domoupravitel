# Инструкции за настройка

## Стъпка 1: Инсталиране на зависимости

```bash
npm install
```

## Стъпка 2: Настройка на Supabase база данни

1. Отворете вашия Supabase проект: https://jraihoszdtzzhwnmpvjx.supabase.co
2. Отидете в **SQL Editor**
3. Копирайте и изпълнете целия SQL скрипт от файла `database.sql`
4. Проверете дали всички таблици са създадени успешно

## Стъпка 3: Настройка на Storage

1. В Supabase Dashboard отидете в **Storage**
2. Създайте нов bucket с име: `documents`
3. Направете bucket-а **публичен** (public)
4. Проверете дали политиките за достъп са настроени правилно

## Стъпка 4: Стартиране на приложението

```bash
npm run dev
```

Приложението ще се отвори на http://localhost:3000

## Стъпка 5: Първо влизане

1. Регистрирайте първия потребител като **Администратор**
2. След това можете да създавате други потребители с различни роли:
   - **Администратор** - Пълен достъп
   - **Редактор** - Може да създава и редактира
   - **Преглед** - Само преглед, без права за промяна

## Важни бележки

- Уверете се, че Row Level Security (RLS) е активиран за всички таблици
- Storage bucket-ът `documents` трябва да е публичен за да могат да се преглеждат файловете
- За production използвайте environment variables за Supabase ключовете

## Build за production

```bash
npm run build
```

Готовите файлове ще бъдат в папката `dist/`.
